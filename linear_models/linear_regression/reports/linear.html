Interactive Scatter Plot Smoothers

Jun 24, 2016

This app demos various one dimensional scatterplot smoothers interactively.

Click on the canvas to add points to the scatterplot. Use the interface below to select a smoothing algorithm and, if needed, hyperparameters for the smoothing.

Once you've added three points, the smoother will kick in automatically.
